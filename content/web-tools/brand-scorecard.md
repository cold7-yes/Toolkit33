## The problem

Brand33 has a proprietary 33-point framework for evaluating creative in the insurance space. Applying it manually took hours per brand, was inconsistent between reviewers, and aged out as soon as the target changed their website or ran a new campaign.

## The solution

A web app that takes a brand, pulls live data about it — website content, visual identity, campaigns, partnerships — and scores it against the full 33-point framework. Output is a structured report that a strategist can walk a client through in minutes instead of days.

## How it works

- Analyzes website content, copy, and visual identity
- Reviews active campaigns and brand partnerships
- Uses live data captured at the moment of the audit
- Scores against all 33 points of the Brand33 framework
- Produces a consistent, comparable report per brand

## Stack

- **Next.js** on Vercel
- **LLM-driven evaluation** against the 33-point rubric
- **Live data capture** from public brand surfaces

## Status

Core scoring engine is functional and producing usable reports. Active refinement on evaluation criteria, UI polish, and adding multi-brand comparison views.
